# Mksite (Fish Shell Web Builder)

`mksite` is een interactieve, razendsnelle webproject-builder geschreven in Fish shell.

## Features
- **Express.js Monolith**: Productie-ready backend met SQLite en foutafhandeling.
- **Vite Front-end**: Bliksemsnelle vanilla front-end template.
- **Vanilla Split**: API en Frontend gescheiden met Concurrently.
- **Premium Full-stack**: React, Tailwind CSS, en een Express/SQLite API gebundeld.

## Installatie
Zet het `mksite.fish` bestand in je `~/.config/fish/functions/` map, en je kunt overal in je terminal het commando `mksite` gebruiken!
